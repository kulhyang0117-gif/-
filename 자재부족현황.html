🤖 sMES 키팅 자동화

1sMES 실행 → 로그인 (SSAT045)
2생산관리 → 조립 자재 kitting → 오늘 날짜 조회
3품목 전체 선택 후 Excel 다운로드 반복
4자재부족현황 로그인 → 키팅 초기화 → 전체 업로드
5실행완료 알림 표시
⚠️ 최초 1회 setup_url_protocol.bat를 관리자로 실행하여 프로토콜을 등록해야 합니다.
