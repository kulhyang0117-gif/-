조립팀 자재현황 시스템 사용 메뉴얼

이 시스템은 생산계획 × BOM × 재고현황을 자동으로 비교 분석하여, 조립 라인별로 어떤 자재가 부족한지 한눈에 파악하고 선제적으로 대응하기 위한 도구입니다.
파일 업로드만 하면 자동으로 부족 자재를 계산하고, Kitting Board · 자재현황 · 포장재 소요량을 모두 확인할 수 있습니다.

포장재 소요량 계산용
📋 Kitting Board 메인 화면
1
라인별 Gantt 차트로 당일 생산 모델과 수량을 시각적으로 확인합니다. 색상으로 자재 준비 상태를 즉시 파악할 수 있습니다.
2
모델 블록 클릭 → 해당 모델의 전체 구성 부품과 재고 현황을 상세 패널에서 확인합니다.
3
부족자재 현황 표에서 라인별 부족 부품 목록, 필요수량, 현재고, 협력사를 한눈에 확인합니다.
4
부족 자재 조달 완료 시 OK 체크박스를 선택하면 해당 행이 녹색으로 표시되어 처리 현황을 관리할 수 있습니다.
◀ ▶ 버튼으로 날짜를 이동하여 일자별 키팅 현황을 확인하세요.
📊 자재현황 상세 분석
1
자재별 부족현황 — 전체 부품 중 부족·위험·충분 상태를 부품코드 단위로 정렬 확인. 긴급도(1일/3일 이내)로 우선순위 파악.
2
모델별 부족현황 — 어떤 모델을 생산할 때 자재가 부족한지 모델 단위로 확인합니다.
3
일자별 부족현황 — 날짜별로 어떤 자재가 언제 부족해지는지 타임라인으로 파악합니다.
4
재고현황 / 모델별 재고현황 — 현재 창고 + 키팅 재고를 품목별·모델별로 확인합니다.
파일 업로드 섹션에서 BOM·생산계획·재고현황 3개 파일 업로드 후 분석 실행 버튼을 클릭하세요.
📦 포장재 필요수량 포장 관리
1
포장재별 집계 — 생산계획 수량 × 포장사양 마스터를 계산하여 포장재 종류별 총 필요수량을 자동 산출합니다.
2
일자별 소요량 — 날짜별로 포장재가 얼마나 필요한지 확인하여 사전 준비 계획을 수립합니다.
3
유형별 / 라인별 소요량 — 박스·트레이·발포지 등 유형별, 조립 라인별로 구분하여 확인합니다.
포장사양 파일을 업로드하지 않으면 포장재 소요량이 계산되지 않습니다.
